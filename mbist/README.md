# MBIST Verification Project

SystemVerilog implementation and verification of a Memory Built-In Self-Test (MBIST) system.

## Features
- SRAM model
- MBIST controller
- Comparator, decoder, counter, and multiplexer modules
- Directed SystemVerilog testbenches
- Timing, area, and power analysis

## Tools Used
- SystemVerilog
- Synopsys VCS
- Synopsys Design Compiler
- SAED32 standard cell library

## Files
- RTL modules: '*.sv'
- Testbenches: 'tb_*.sv'
- Constraints: 'bist_constraint.sdc'
- Synthesis script: 'script.tcl'
- Reports: 'area.rpt', 'timing.rpt', 'power.rpt'
